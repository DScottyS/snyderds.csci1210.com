More HTML

in this assignment, we'll look at more HTML, including headings, divs, paragraphs, and lists

We can, and often do, nest div elements within one another. The reasons for doing this will become apparent as wee go on, but divs are "general purpose" kinds of elements that can be customized for different uses (more on that when we get into Cascading Style Sheets (CSS)).

Another common set of HTML elemets are lists: Ordered, Unordered, and Definition Lists.

Lists

Ordered Lists

Ordered lists are also known as "Numbered Lists." Sometimes, the order in which items appear in a list is important, like a list of instructions.

  1. Underwear
  2. Pants
  3. Shoes

It's important to remember that with numbered lists, we don't have to add the numbers. The browser will do that when the list is rendered.

Unordered Lists

Another type of lists, for when the ordere of the list items isn't important, is called "Unordered Lists." We sometimes refer to these as bulleted lists

  • Eat
  • Sleep
  • Code

Definition Lists

A final type of lists included in HTML is "Definition Lists" These can be useful for glossaries, phone lists, etc.
